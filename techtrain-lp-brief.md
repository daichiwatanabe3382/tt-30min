# TechTrain Direct LP制作ブリーフ

## 概要

新卒エンジニア採用に困っている人事担当者向けのLP制作。
オーガニック検索からの流入を想定した直接訴求型。

### ゴール
- 資料請求
- 無料相談の申し込み

### 対象サービス
- スカウト（DirecTrain）
- ヘッドハント（ダイレクトエージェント）
※求人広告は対象外

---

## LP構成

### 1. ファーストビュー

**キャッチコピー：**
```
人物理解から始まる、新しいエンジニア採用
```

**サブコピー：**
```
スキル × 経歴 × 人物評価。3軸で立体的に候補者を事前に確認できる
だから「この人と働きたい」が見つかるダイレクトスカウト
```

**数字訴求（4つ並列）：**

| 数字 | コピー |
|------|--------|
| 返信率30%超 | 業界平均の3〜6倍 |
| 週1時間で運用 | スカウト代行で工数削減 |
| 代行月5万円〜 | 定額で代行は業界唯一 |
| 13,000名登録 | メンター評価済みの母集団 |

**CTA：**
- 資料をダウンロード
- 無料相談を予約する

### 2. 課題提起セクション

**見出し：**
```
「スキルは良さそう」、でも会ってみたら違った——そんな経験ありませんか？
```

**課題リスト（6つ）：**

| 課題 | 補足コピー |
|------|-----------|
| プロフィールだけでは人柄がわからない | GitHubやポートフォリオは見た。でも一緒に働きたいかは別の話 |
| 会ってから「合わない」がわかる | 5分で違うと感じても、1時間の面談は続く |
| カルチャーマッチを重視したいのに、判断材料がない | スキルは絞れる。でもマインドは会わないとわからない |
| スカウト運用に手が回らない | 候補者ピック、文面作成、送信、返信対応…工数が膨大 |
| 他業務が忙しく、採用に集中できない | 面接調整、説明会準備、内定者フォロー。スカウトは後回しになりがち |
| 現場エンジニアを巻き込む余裕がない | 協力を頼みたいが、本業で手一杯と言われる |

### 3. Whyセクション

**見出し：**
```
TechTrain Directは「一緒に働きたい人」を採用するためのエンジニア採用サービスです
```

**リード文：**
```
能力の高さももちろん大切ですが、それよりも大切なことは"一緒に働きたい"と思える人材かどうかにあります。
```

**根拠（代表メッセージ）：**
```
TechTrainの原点は、CyberAgent 藤田晋会長からの教えである

「能力の高い人ではなく一緒に働きたいひとを採用する」
「すべての事業の仕入れは採用にある」

にあります。

この理想的な採用を、エンジニア採用に悩むすべての組織に提供する。
これを実現するために TechTrain を立ち上げました。
```

**代表プロフィール（右側に配置）：**
- 代表取締役CEO 小澤 政生
- 2010年京都大学経済学部経営学科卒業。株式会社サイバーエージェントにて採用責任者を担当。7年間で約15,000人の学生と面談。


### 5. サービス特徴セクション

**見出し：**
```
TechTrain Directで実現できること
```

**3つの特徴：**

| # | 見出し | 説明 |
|---|--------|------|
| ① | スキル以外の「キャラクター」が見える | メンターが記録した人物評価・推薦コメントを閲覧可能。「論理的思考が強い」「素直で吸収が早い」など、プロフィールだけではわからない人柄が事前に見えます。 |
| ② | メンターのスキル評価・推薦付きでアプローチ | 開発組織を作るプロ（CTO、VPoE、EM経験者）が技術力とマインドを多角的に評価済み。第三者の「お墨付き」があるから、安心してスカウトを送れます。 |
| ③ | CS伴走型で、承諾まで一緒に走る | 求人票の打ち出し設計、スカウト文面のFB、ターゲットチューニング、返信後のフォローまで。専属CSがPDCAを回しながら、採用成功まで伴走します。 |

### 6. サービスラインセクション

**見出し：**
```
採用スタイルに合わせて選べる2つのプラン
```

#### 6-1. スカウト（DirecTrain）

**タグライン：**
```
週1時間で、いい人材との面談が実現
```

**ポイント：**
- 候補者ピック・メッセージ作成を代行
- 貴社はスカウト確認＆送信だけでOK
- 返信率30%超（業界平均の3〜6倍）

**料金：**
```
定額プラン 月25万円〜 / 成果報酬プランあり
```

**補足：**
```
定額でスカウト代行ができるのはTechTrainだけ
```

#### 6-2. ヘッドハント（ダイレクトエージェント）

**タグライン：**
```
会いたい人を狙い撃ち。成果報酬なし
```

**仕組み（ステップ形式）：**

| STEP | 内容 |
|------|------|
| ① 面談枠を購入 | 前払いで必要な枠数だけ購入（5枠〜） |
| ② 選抜リストを作成 | 面談枠の2〜3倍を目安に、貴社にマッチしそうな人材を弊社で選抜 |
| ③ 1時間の打ち合わせ | 選抜メンバーの概要をお伝え |
| ④ 会いたい人を選ぶ | 「会いたい」「会いたくない」をジャッジするだけでOK |
| ⑤ 面談をセット | 会いたい人を弊社が口説き、確実に面談できるよう支援 |

**料金：**
```
面談1枠 40万円 × 5枠〜（内定承諾の成果報酬ゼロ）
```

### 7. 選ばれる理由セクション

**見出し：**
```
TechTrainが選ばれる理由
```

**5つの理由：**

| # | 理由 | 説明 |
|---|------|------|
| ① | 感度の高い13,000名の母集団 | 広告ではなくメンター経由の口コミで集まった成長意欲の高いエンジニア |
| ② | 開発組織を作るプロの評価 | Microsoft、メルカリ、サイバーエージェント等出身。30%以上がCxO経験者 |
| ③ | 唯一の「定額×代行」モデル | 他社は成果報酬 or 代行なし。両方できるのはTechTrainだけ |
| ④ | 採用の「こみみ情報」をお届け | 候補者の志望業界、キャラクター、面談後のリアルな感想。中間の立場だから伝えられる本音をフィードバック。「なぜ御社を選ばなかったか」も正直にお伝えし、次の採用に活かせます |
| ⑤ | AIスカウト、学生にはバレてます | だから私たちは「納品して終わり」の外注ではなく、貴社の採用チームの一員として入り込みます。候補者理解、文面作成、細かな修正依頼、返信後のフォローまで。痒いところに手が届く伴走型で、承諾まで一緒に走り切ります |

## デザイン参考

### ブランドカラー
- メイン：紺（TechTrainのロゴカラー）
- アクセント：黄色
- 背景：白

### トンマナ
- 信頼感のあるビジネストーン
- でも堅すぎない、親しみやすさも
- 「伴走」「一緒に」のウェットな温度感

### 参考にすべき既存デザイン
- 営業資料のスライドデザイン（黄色×紺のカラーリング）
- キャラクター（猫のイラスト）を適宜使用

---

## 実装要件

- レスポンシブ対応（SP/PC）
- CTAボタンは各セクション終わりにも配置を検討
- ファーストビューのCTAは目立たせる
- 数字訴求は視認性高く
- 図解（2軸→3軸）はビジュアルで表現
